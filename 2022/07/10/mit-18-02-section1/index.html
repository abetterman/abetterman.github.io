Multivariable Calculus | Vectors

Author: Zhang Qijie Date: July 10, 2022  14:18:01 Category: mit, calculus,math

这是我重修数学计划的第二门课程：MIT 18.02 多变量微积分的学习笔记。

本课程涵盖向量和多变量微积分，包括向量和矩阵、偏导数、二重和三重积分，以及 2 /3 维空间中的向量微积分。

Vectors

一个向量 (notation: $\vec{A}$ ) 具有方向和长度 ( $|\vec{A}|$ ).

关于向量，一个重要的观点是：我们可以几何的 (Geometric) 和代数的 (Algebraic) 两种不同的方式来看向量。

Geometric View

向量具有大小 (magnitude) 和方向 (direction)，两个具有相同的大小和方向的向量被认为是同一个向量。

向量 $\overrightarrow{\mathbf{P Q}}$, P 称为向量的 tail， Q 称为向量的 head

向量 A 的大小 (magnitude) 表示为：$|\vec{A}|$，向量的大小即向量的长度。

Scaling, adding and subtracting vectors

缩放矢量意味着通过比例因子改变其长度。例如：

向量相加即两个向量的头尾 (head to tail) 相连：$A+B$ 或者 $B + A$

将向量想象为位移 (displacements) 是很有用的，则 $A + B$ 可以想象为 位移 A，然后再位移 B。

向量相减即两个向量尾尾相连 (tail to tail)，$A - B$ 即 $A + (-B)$

同理，$A - B$ 可以想象为从 B 的 head 位移到 A 的 head

Algebraic View

在 $xy-plane$ 中， 向量 $A$ 可分解为 $A = a_1i + a_2j$， $i$ 为 $x$ 轴上的分量，$j$ 为 $y$ 轴上的分量， $a_1,a_2$ 是 scalars。

Vector algebra using Coordinates

向量 $A= a_1i + a_2j$ ，向量 $B= b_1i + b_2j$

$A$ 的大小： $|\mathbf{A}|=\sqrt{a_{1}^{2}+a_{2}^{2}}$

相加 $A + B = (a_1 + b_1)i + (a_2 + b_2)j$

相减 $A - B = (a_1 - b_1)i + (a_2 - b_2)j$

无论是 2 维的平面，还是 3 维的空间，向量的代数运算规则都是一样的。

比如： $A = a_1i + a_2j + a_3k$ ，$B = b_1i + b_2j + b_3k$

$|\mathbf{A}|=\sqrt{a_{1}^{2}+a_{2}^{2}+a_{3}^2}$

相加 $A + B = (a_1 + b_1)i + (a_2 + b_2)j + (a_3 + b_3)k$

相减 $A - B = (a_1 - b_1)i + (a_2 - b_2)j + (a_3 - b_3)k$

Unit Vectors

单位向量，即任何具有单位长度的向量。表示为 $\hat u$

$x,y,z$ 三个轴的单位向量：$i, j, k$

任何向量都可以 rescale 成单位向量，如：

求与向量 $<3, 4>$ 平行的单位向量：

$|<3, 4>| = 5$，所以 $\frac{1} {5} <3, 4>$ ，即 $<\frac {3} {5}, \frac {4} {5}>$ 是平行于 $<3, 4>$ 的单位向量。

即 向量 $a = λb$, 则 向量 $a, b$ 平行。

Vector Problems

  1. 一条河流以每小时 3 英里的速度流动，而划船者以每小时 6 英里的速度流动。 划船者应该采取什么方向直接过河？

  2. 如果河流以每小时 6 英里的速度流动，而划船者以每小时 3 英里的速度流动，请回答相同的问题。

第一个问题，即是求上图中的角 $\theta$ ，net velocity 是向量 1：小船速度与向量 2：河流速度的相加；小船的过河方向为垂直于河流。所以：

​ $\sin (\theta)=\frac{3}{6} \Rightarrow \theta=30^{\circ}=\pi / 6$

第二个问题，

​ $\sin (\theta)=\frac{6}{3}$，显然是不可能的（$\sin (\theta) \leq 1$）

  1. 求向量 $<2, 3>$ 平行的单位向量

​ $\widehat{\mathbf{u}}=\frac{\langle 2,3\rangle}{\sqrt{13}}=\left\langle\frac{2}{\sqrt{13}}, \frac{3}{\sqrt{13}}\right\rangle$

  1. 用向量证明平行四边形的对角线相互平分

这个问题即是求线段 AC 的中点与线段 BD 的中点是同一个点。

设 $M_1$ 是 AC 的中点，$M_2$ 是 BD 的中点。证明 $M_1 = M_2$ ，即证明 $\overrightarrow{\mathbf{A M}_1}=\overrightarrow{\mathbf{A M}_2}$

很明显， $\overrightarrow{\mathbf{A M}_{1}}=\frac{1}{2} \overrightarrow{\mathbf{A C}}$ ，同时， $ABCD$ 是平行四边形，由此，我们得到：

​ $\overrightarrow{\mathbf{A B}}=\overrightarrow{\mathbf{D C}} \Rightarrow \overrightarrow{\mathbf{A B}}=\frac{1}{2} \overrightarrow{\mathbf{A B}}+\frac{1}{2} \overrightarrow{\mathbf{A B}}=\frac{1}{2} \overrightarrow{\mathbf{A B}}+\frac{1}{2} \overrightarrow{\mathbf{D C}}$

利用 $A,B,C,D$ 改写 $AM_{2}$ :

$\overrightarrow{\mathbf{A M}_{2}}=\overrightarrow{\mathbf{A B}}+\frac{1}{2} \overrightarrow{\mathbf{B D}}=\frac{1}{2} \overrightarrow{\mathbf{A B}}+\frac{1}{2} \overrightarrow{\mathbf{D C}}+\frac{1}{2} \overrightarrow{\mathbf{B D}}=\frac{1}{2}(\overrightarrow{\mathbf{A B}}+\overrightarrow{\mathbf{B D}}+\overrightarrow{\mathbf{D C}})=\frac{1}{2} \overrightarrow{\mathbf{A C}}$

以上，已经证明 $\overrightarrow{\mathbf{A M}_1} = \overrightarrow{\mathbf{A M}_2}$

  1. 用向量法证明空间四边形的中点形成一个平行四边形

    此四边形的四条边分别用 $A B C D$ 表示，则有 $A + B = C + D$，如图所示：

    上图中，向量 $\overrightarrow{\mathbf{A C}} = \frac {1}{2} C - \frac {1}{2} A$ ，同理， $\overrightarrow{\mathbf{B D}} = \frac {1}{2} B - \frac {1} {2} D$

    需要证明 $\overrightarrow{\mathbf{A C}} = \overrightarrow{\mathbf{B D}}$

    因为 $A + B = C + D$，所以， $C - A = B - D \Rightarrow \frac {1}{2}(C-A)= \frac {1}{2}(B-D)$，即 $\overrightarrow{\mathbf{A C}} = \overrightarrow{\mathbf{B D}}$

  2. Find $A + B + C$

    向量就是位移 (displacements)，$A + B + C$ 即是跟随 A 的净位移，然后位移到 B，然后位移到 C。

  3. 三角形的中线是一个顶点到对边中点的向量。证明三角形的中位线之和 $= 0$

即证明 $\overrightarrow{\mathbf{C P}} + \overrightarrow{\mathbf{ B Q}} + \overrightarrow{\mathbf{A R}} = 0$

$P = \frac {1}{2} (A + B) \Rightarrow \overrightarrow{\mathbf{C P}} = \frac {1}{2} (A + B) - C$

同理，$\overrightarrow{\mathbf{B Q}} = \frac {1}{2} (A + C) - B$，$\overrightarrow{\mathbf{A R}} = \frac {1}{2} (B + C) - A$

$\overrightarrow{\mathbf{C P}} + \overrightarrow{\mathbf{ B Q}} + \overrightarrow{\mathbf{A R}} = (\frac {1}{2} (A + B) - C) + (\frac {1}{2} (A + C) - B) + (\frac {1}{2} (B + C) - A) = 0$
