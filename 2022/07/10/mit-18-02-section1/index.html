Author: Zhang Qijie Date: July 10, 2022  14:18:01 Category: mit, calculus,math

这是我重修数学计划的第二门课程：MIT 18.02 多变量微积分的学习笔记。

本课程涵盖向量和多变量微积分，包括向量和矩阵、偏导数、二重和三重积分，以及 2 /3 维空间中的向量微积分。

Vectors

一个向量 (notation: $\vec{A}$ ) 具有方向和长度 ( $|\vec{A}|$ ).

关于向量，一个重要的观点是：我们可以几何的 (Geometric) 和代数的 (Algebraic) 两种不同的方式来看向量。

Geometric View

向量具有大小 (magnitude) 和方向 (direction)，两个具有相同的大小和方向的向量被认为是同一个向量。

向量 $\overrightarrow{\mathbf{P Q}}$, P 称为向量的 tail， Q 称为向量的 head

向量 A 的大小 (magnitude) 表示为：$|\vec{A}|$，向量的大小即向量的长度。

Scaling, adding and subtracting vectors

缩放矢量意味着通过比例因子改变其长度。例如：

向量相加即两个向量的头尾 (head to tail) 相连：$A+B$ 或者 $B + A$

将向量想象为位移 (displacements) 是很有用的，则 $A + B$ 可以想象为 位移 A，然后再位移 B。

向量相减即两个向量尾尾相连 (tail to tail)，$A - B$ 即 $A + (-B)$

同理，$A - B$ 可以想象为从 B 的 head 位移到 A 的 head

Algebraic View

在 $xy-plane$ 中， 向量 $A$ 可分解为 $A = a_1i + a_2j$， $i$ 为 $x$ 轴上的分量，$j$ 为 $y$ 轴上的分量， $a_1,a_2$ 是 scalars。

Vector algebra using Coordinates

向量 $A= a_1i + a_2j$ ，向量 $B= b_1i + b_2j$

$A$ 的大小： $|\mathbf{A}|=\sqrt{a_{1}^{2}+a_{2}^{2}}$

相加 $A + B = (a_1 + b_1)i + (a_2 + b_2)j$

相减 $A - B = (a_1 - b_1)i + (a_2 - b_2)j$

无论是 2 维的平面，还是 3 维的空间，向量的代数运算规则都是一样的。

比如： $A = a_1i + a_2j + a_3k$ ，$B = b_1i + b_2j + b_3k$

$|\mathbf{A}|=\sqrt{a_{1}^{2}+a_{2}^{2}+a_{3}^2}$

相加 $A + B = (a_1 + b_1)i + (a_2 + b_2)j + (a_3 + b_3)k$

相减 $A - B = (a_1 - b_1)i + (a_2 - b_2)j + (a_3 - b_3)k$

Unit Vectors

单位向量，即任何具有单位长度的向量。表示为 $\hat u$

$x,y,z$ 三个轴的单位向量：$i, j, k$

任何向量都可以 rescale 成单位向量，如：

求与向量 $<3, 4>$ 平行的单位向量：

$|<3, 4>| = 5$，所以 $\frac{1} {5} <3, 4>$ ，即 $<\frac {3} {5}, \frac {4} {5}>$ 是平行于 $<3, 4>$ 的单位向量。

即 向量 $a = λb$, 则 向量 $a, b$ 平行。

    这个问题即是求线段 AC 的中点与线段 BD 的中点是同一个点。

    设 $M_1$ 是 AC 的中点，$M_2$ 是 BD 的中点。证明 $M_1 = M_2$ ，即证明 $\overrightarrow{\mathbf{A M}_1}=\overrightarrow{\mathbf{A M}_2}$

    ​ $\overrightarrow{\mathbf{A B}}=\overrightarrow{\mathbf{D C}} \Rightarrow \overrightarrow{\mathbf{A B}}=\frac{1}{2} \overrightarrow{\mathbf{A B}}+\frac{1}{2} \overrightarrow{\mathbf{A B}}=\frac{1}{2} \overrightarrow{\mathbf{A B}}+\frac{1}{2} \overrightarrow{\mathbf{D C}}$

    $\overrightarrow{\mathbf{A M}_{2}}=\overrightarrow{\mathbf{A B}}+\frac{1}{2} \overrightarrow{\mathbf{B D}}=\frac{1}{2} \overrightarrow{\mathbf{A B}}+\frac{1}{2} \overrightarrow{\mathbf{D C}}+\frac{1}{2} \overrightarrow{\mathbf{B D}}=\frac{1}{2}(\overrightarrow{\mathbf{A B}}+\overrightarrow{\mathbf{B D}}+\overrightarrow{\mathbf{D C}})=\frac{1}{2} \overrightarrow{\mathbf{A C}}$

    以上，已经证明 $\overrightarrow{\mathbf{A M}_1} = \overrightarrow{\mathbf{A M}_2}$

Dot Product

向量的点积，定义：$\vec A \cdot \vec B = a_1b_1 + a_2b_2 + a_3b_3$ (Scalar, not a vector)

定理：点积的几何意义：

​ $\vec A \cdot \vec B = |\vec A||\vec B|cos\theta$

证明：

向量 $\vec A \cdot \vec A = |\vec A|^2cos0 = |\vec A|^2$

一个三角形的三边：$A, B, C$，$\vec C=\vec A-\vec B$ ，根据余弦定理：$|\vec C|^2 = |\vec A|^2 + |\vec B|^2 - 2|\vec A||\vec B|cos\theta$

得到： $|\vec C|^2 = \vec C \cdot \vec C = (\vec A - \vec B)(\vec A - \vec B) = |\vec A|^2 + |\vec B|^2 - 2 \vec A \cdot \vec B$

可见，点积定理是余弦定理的向量版本。

Components and Projection

$\vec A$ 是一个向量，$\hat u$ 是一个单位向量，则 $\vec A$ 在 $\hat u$方向上的分量为 $\vec A \cdot \hat u$

​ $\vec A \cdot \hat u = |\vec A||\hat u|cos\theta$ [Note: the component is a Scalar]

平行于 $\hat u$ 的边，也称为 $\vec A$ 在 $\hat u$ 上的正交投影 (orthogonal projection)

对于非单位向量，比如 $\vec A$ 在向量 $\vec B$ 方向的分量为 $\vec A$ 在 $\hat u = \frac {\vec B} {|\vec B|}$ 方向的分量。($\hat u$ 与向量 $\vec B$ 相同方向)

Example: find the component of $A$ in the direction of $B$

  1. $|A| = 2, |B| = 5, \theta = \frac {\pi} 4$

    component = $|A|cos\theta = 2cos(\pi/4) = \sqrt2$

  2. $A= i + 2j, B=3i + 4j$

    Unit vector in direction B: $\hat u = \frac B {|B|} = \frac 3 5 i + \frac 4 5 j \Rightarrow$ component is: $A \cdot \frac B {|B|} = \frac 3 5 + \frac 8 5 = \frac {11} 5$

  3. find the component of $A=\langle 2,2 \rangle$ in the direction of $\hat u= \langle -1,0 \rangle$

    the componet is $A \cdot \hat u=\langle 2,2 \rangle \cdot \langle -1,0 \rangle = -2$

We emphasize one more time that the component of a vector is a scalar.

Expansions along the wrong row or column

一个有趣且有用的事实是，尽管拉普拉斯扩展:

​ $\sum_{j=1}^{K} A_{i j} C_{i j}=\operatorname{det}(A)$

我们得到：

​ $\sum_{j=1}^{K} A_{i j} C_{k j}=0$ （当 $k \neq i$ ）

In other words, if we multiply the elements of row $i$ with the cofactors of a different row $k$ and we add them up, we get zero as a result.

Cofactor Matrix

Definition:

​ Let $A$ be a $K \times K$ matrix. Denote by $C_{ij}$ the cofactor of $A_{ij}$. Then the $K \times K$ matrix $C$ such that its $(i,j)-th$ entry is equal to $C_{ij}$ for every $i$ and $j$ is called cofactor matrix of $A$.

Adjoint Matrix

adjoint matrix 是 cofactor matrix 的转置．

Definition:

​ Let $A$ be a $K \times K$ matrix and $C$ its cofactor matrix. The adjoint matrix of $A$, denote by $adj(A)$, is:

​ $adj(A) = C^T$

Adjoint, determinant and inverse

命题：

​ Let $A$ be a $K \times K$ matrix and $adj(A)$ its adjoint. Then,

​ $Aadj(A) = adj(A)A = det(A)I$

命题：
